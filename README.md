# 🏐 Благотворителен волейболен турнир „В подкрепа на Пепи“ ❤️

Live tracking система за резултати, класиране и снимки — всички виждат всичко в реално време през общ линк.

## Възможности
- **Програма** — 32 мача на 2 игрища, „кой играе сега / кой е следващ“ на живо
- **Live класиране** — 4 групи, обновява се в реално време на всички устройства (Supabase Realtime)
- **Плейофи** — топ 2 от група, анимиран случаен жребий, пълна схема до финала + мач за 3-то място
- **📸 Live галерия** — всеки може да качва и разглежда снимки от турнира в реално време
- **📤 Share карти** — красива картинка с резултата за споделяне след всеки мач
- **⏱ Отброяване** — countdown до първия сервис на началния екран
- **🔑 Режим организатор** — резултати, жребий и нулиране са защитени с PIN (проверяван на ниво база през RLS); зрителите само гледат и качват снимки

## Правила
- Групова фаза: сет до 15 т. · Плейофи: сет до 21 т., трети сет до 15 т.
- Сет се печели с минимум 2 точки разлика (напр. 17:15, 23:21)
- Победа 2:0 → 3 т. · Победа 2:1 → 2 т. / 1 т. за загубилия

## Стек и деплой
Един HTML файл без build стъпка + Supabase (Postgres + Realtime + Storage, free tier).
Деплой в Netlify: Import from GitHub, без build настройки, publish directory root.
При липса на интернет приложението показва последно заредените данни (localStorage кеш).
