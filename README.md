# 🏐 Благотворителен волейболен турнир „В подкрепа на Пепи“

Live tracking система за резултати и класиране — групова фаза (4 групи, 2 игрища) + плейофи с анимиран жребий.

## Правила
- Групова фаза: сет до 15 т., плейофи: сет до 21 т. (трети сет до 15 т.)
- Сет се печели с минимум 2 точки разлика
- Победа 2:0 → 3 т. · Победа 2:1 → 2 т. / 1 т. за загубилия

## Стартиране
Статично приложение без зависимости — отвори `index.html` в браузър или деплой в Netlify (няма нужда от build настройки, publish directory: root).

Данните се пазят в localStorage на устройството. Има Export/Import за бекап.
